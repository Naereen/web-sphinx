À propos
--------
Cette page accueille une liste des problèmes à résoudre.

Cette fonctionnalité viend de l'extension officielle ``sphinx.ext.todo``.

De tels soucis peuvent être déclarés via : ::

    .. todo::

    Voici un soucis.
