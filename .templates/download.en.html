Download these pages in PDF

These web pages are generated with Sphinx. And when I generate them, one of my script use rst2pdf to also produce a PDF version of each HTML pages.

The folder pdf/ is there to download some PDF files produced that way.

This is already functionnal, but some pages are useless without JavaScript (not yet integrated in the PDF documents, but it might be possible).


Download folder

The folder dl/ is there to download backup archives of somes projects.

This folder is protected by a username and a password. It is possible to ask for one by email, if needed, to naereen[@]crans[.]org.

Sources for this site

"Static" content : rST files

The git repository lbesson/web-sphinx hosts rST files, Jinja templates, and many others ressources required to generate these web pages with sphinx.
This site is an open-source and a free software project. A zip archive can be downloaded with this link.

"Dynamical" content : JavaScript files

The git repository lbesson/web-sphinx-scripts hosts JavaScript files, required by these web pages to be fully functional.
These scripts are written by me, or are from other open-source projects. I think every used project is freely distributed under a license free nice enough to "allow me" to include them in my repository, which can be downloaded with this zip archive.
Contact me if I'm wrong about this, and if that bothers you !

Troubles

If you encounter any difficulty, or if you have a question, please use the page web-sphinx/issues.
