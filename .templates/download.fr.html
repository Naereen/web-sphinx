Télécharger ces pages en PDF

Ces pages web sont générées avec Sphinx. Et lors de leurs générations, un de mes scripts produit aussi une version PDF de certaines pages (avec rst2pdf).

Le dossier pdf/ est là pour vous permettre de télécharger les fichiers PDF ainsi produits.

Ceci fonctionne, mais certaines pages perdent leur intérêt (le JavaScript n'est pas encore intégré dans les documents PDF produits, bien que ce soit peut-être possible).


Dossier de téléchargement

Le dossier dl/ est là pour télécharger des archives, et des documents particuliers.

L'accès à ce dossier nécessite un nom d'utilisateur et un mot de passe. Il est possible d'en demander un par email, si nécessaire, à l'adresse suivante naereen[@]crans[.]org.

Sources de ce site

Contenu "statique" : des fichiers rST

Le dépôt git lbesson/web-sphinx héberge les fichiers rST, les templates, et autres ressources nécessaires à la génération de ces pages web avec sphinx.
Ce site est un projet libre et open-source. Une archive peut être récupérée via ce lien.

Contenu "dynamique" : des scripts JavaScript

Le dépôt git lbesson/web-sphinx-scripts héberge les scripts JavaScript, nécessaires au bon fonctionnement de ces pages web.
Ces scripts sont écrits par moi ou issus de projets open-source. Normalement, chaque projet utilisé est distribué sous license libre, suffisamment permissive pour "m'autoriser" à les redistribuer via ce dépôt, ou cette archive.
Contactez-moi si je me trompe à ce propos, et si ça vous dérange !

Problèmes

Si vous rencontrez une difficulté quelconque ou que vous avez des questions, veuillez utiliser la page web-sphinx/issues.
