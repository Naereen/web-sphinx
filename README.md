Requirements ?
--------------

This project use the following software :

 * LaTeX (pdflatex & hevea);
 * GNU Make;
 * git;
 * rsynx or scp;
 * ack-grep or grep;
 * Sphinx;
 * GNU Plot;
 * GPG;
 * GraphViz;
 * rst2pdf, rst2html, rst2s5;
 * and the following Sphinx contrib extensions :

    * cheeseshop;
    * gnuplot;
    * autorun;
 * (optional) ocaml, octave, to embed examples.
