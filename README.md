Requirements ?
--------------

This project use the following software :

 * LaTeX (pdflatex & hevea);
 * GNU Make;
 * git;
 * rsynx or scp;
 * ack-grep or grep;
 * Sphinx;
 * rst2pdf, rst2html, rst2s5;
 * the following Sphinx contrib extensions :

    * cheeseshop;
    * gnuplot;
    * autorun;
 * (optional) ocaml, octave, to embed examples.
